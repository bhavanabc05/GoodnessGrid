About Goodness Grid

A Network of Good Deeds Connecting Communities

🎯 The Problem

Every day, tons of food, clothes, medicines, and essential items are discarded while many underprivileged communities continue to suffer from shortages. NGOs, orphanages, old-age homes, and many others often rely on local contacts or manual donations, which are not systematic or sustainable.

💡 Our Solution

Goodness Grid is a centralized online platform that connects donors and receivers efficiently. We bridge the gap between surplus and need through technology.

🎁
For Donors

Individuals, restaurants, companies, and pharmacies can easily donate surplus items

🏥
For Receivers

NGOs, orphanages, schools, hospitals can browse and claim needed items

🚚
For Volunteers

Community members help with pickup and delivery logistics

✨ Key Features

  • ✅ Multi-category donations (food, clothes, books, medicine, electronics)
  • ✅ Real-time posting and claiming system
  • ✅ Search and filter functionality
  • ✅ Role-based dashboards
  • ✅ Volunteer assignment and tracking
  • ✅ Admin monitoring and user verification
  • ✅ Complete transaction lifecycle management

🛠️ Technology Stack

Frontend
  • HTML5, CSS3
  • Bootstrap 5
  • Jinja2 Templates
Backend
  • Python 3
  • Flask Framework
  • MySQL Database
  • Werkzeug Security
